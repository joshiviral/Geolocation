# Geolocation

This web application depicts about tracking the users location. To be more specific, it will utlimately throw the IP address of the users system
based on when the user access the web application. Here using MYSQL database, the nearby ip adresses around the areas are being stored.
So, when the user access the application, the system will throw the ip address of the users area from where the application is being accessed
and on database it will compare with the predefined IP address based on the areas being stored and if it matches with the application,
the area name of the location will be displayed on the system


Technologies- PHP, 3rd Party APIs
